# 证书的结构

## 结构图

```text
+-----------------------------------------------------------------------+
|                         X.509 Certificate                             |
+-----------------------------------------------------------------------+
|  [TBSCertificate] (To Be Signed - 证书主体内容)                       |
|  ├── Version (版本号: V1, V2, 或 V3)                                  |
|  ├── Serial Number (证书序列号)                                       |
|  ├── Signature Algorithm Identifier (签名算法标识)                    |
|  ├── Issuer (颁发者名称)                                              |
|  ├── Validity (有效期: Not Before / Not After)                       |
|  ├── Subject (使用者/持有者名称)                                       |
|  ├── Subject Public Key Info (使用者公钥信息)                         |
|  │    ├── Public Key Algorithm (公钥算法)                             |
|  │    └── Subject Public Key (公钥明文)                               |
|  ├── Issuer Unique ID (颁发者唯一ID - V2/V3可选)                      |
|  ├── Subject Unique ID (使用者唯一ID - V2/V3可选)                     |
|  │                                                                    |
|  └── ──► [Extensions] (扩展字段(以下为常用字段) - 仅在 V3 版本中存在)                  |
|           ├── Basic Constraints (基本限制)                            |
|           ├── Key Usage (密钥用法)                                    |
|           ├── Authority Key Identifier (密钥用法)                                    |
|           ├── Subject Alternative Name (SAN - 使用者可选名称)          |
|           │                                                           |
|           ├── ──► [Authority Information Access (AIA)] (包含多种，这里列举其中2种) |
|           │        ├── Access Method: On-line Certificate Status (OCSP)|
|           │        │   └── URI: http://ocsp.example.com               |
|           │        └── Access Method: Certification Authority Issuer  |
|           │            └── URI: http://crt.example.com/ca.crt         |
|           │                                                           |
|           └── CRL Distribution Points (CDP - 证书吊销列表分发点)      |
+-----------------------------------------------------------------------+
|  Signature Algorithm Identifier (证书签名算法)                        |
+-----------------------------------------------------------------------+
|  Signature Value (CA 对 TBSCertificate 的数字签名值)                  |
+-----------------------------------------------------------------------+
```
